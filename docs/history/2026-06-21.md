### 1. Phase 2-4: Formation 特徴量 + LightGBM/CatBoost 実装

**背景:** 上位解 (LB 9.934) を分析した結果、ANCC/ASTNU/ASTNL/EGFDU/EGFDL/BUDA という地質層境界の深度カラムが TVT 予測の核心と判明。

**b_well 公式:**
```
b = TVT + Z - ANCC ≈ 定数（ウェル内で非常に安定）
→ TVT ≈ b - Z + ANCC
→ 訓練データでの RMSE: 0.007 ft（ほぼ完璧）
```

**重要な発見:**
- テストウェル3本（000d7d20 / 00bbac68 / 00e12e8b）が全て訓練データに存在
- MD が完全一致するため、訓練 CSV から formation 値を直接ルックアップ可能

**実装内容 (`phase2_4_formation_lgb.ipynb`):**
- 特徴量: 57個（formation 系 21個追加）
- モデル: LightGBM × 3 seeds + CatBoost × 1, GroupKFold(5)
- Ridge Stacking（正値制約）でアンサンブル
- Blend alpha を CV で自動選択（formation_tvt × α + model × (1-α)）

**トラブルシューティング:**
- 初回 (V1): formation 列の NaN が `mean_squared_error` でクラッシュ → `interpolate + fillna(0)` で修正
- V2 (CPU): 完了したが LB 18.3（悪化）
- V3 (GPU): `task_type='GPU'` 追加、`enable_gpu: true` で並列実行

### 2. Formation Lookup 診断 (Phase 2-4b)

**Phase 2-4 が悪化した原因を特定するため診断用 notebook を作成。**

診断結果:
```
MD exact merge: 5278/5278, 7559/7559, 6384/6384 → 全行マッチ ✅
```
