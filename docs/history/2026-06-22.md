## やったこと

### 1. Kaggle API Bearer 認証対応

CLI (`kaggle kernels push`) が 401 Unauthorized になる問題を調査。  
原因: `.zshrc` の `KAGGLE_TOKEN` に新形式 `KGAT_` トークンがあるが、
新形式は Basic 認証でなく **Bearer 認証** が必要。CLI が未対応。

**対応:** Python `requests` で直接 push する関数を実装。
```python
TOKEN = "KGAT_..."
HEADERS = {"Authorization": f"Bearer {TOKEN}", "Content-Type": "application/json"}
r = requests.post("https://www.kaggle.com/api/v1/kernels/push", headers=HEADERS, json=body)
# 既存カーネル更新: body に "id": kernel_id (整数) を含める
```
