### 2. 方針転換: lb7159 フォークへ

**背景:**

romantamrazov ベースで 10.3 台に留まっているのは、以下の要素が根本的に欠けているため:

| 欠けている要素 | romantamrazov | lb7159 (LB 7.159) |
|---|---|---|
| b_well バリアント | b_wls のみ | full/early/mid/late/wls × 6地層 |
| PF ensemble | 7 beams, 1 seed PF | 128 seeds × 4 scales |
| 空間事前分布 | IDW (ANCC のみ) | FormationPlaneKNN + DenseANCCImputer |
| Pipeline 数 | 1 (LGB+CB+Ridge) | Dual (Pipeline A + Pipeline B) |
| Gold overlay | なし | あり (conservative プロファイル) |

romantamrazov に個別追加するより、lb7159 をフォークして XGBoost を上乗せする方が大幅に効率的。

### seg_b_well（b_well 5バリアント）

```python
def seg_b_well(ktvt, kz, form_col):
    bv = ktvt + kz - form_col  # TVT + Z - formation = constant
    b_full = median(bv)           # 全点のmedian
    b_early = median(bv[:n//3])   # 前1/3
    b_mid = median(bv[n//3:2n//3])# 中1/3
    b_late = median(bv[-50:])     # 末尾50点
    b_wls = exp_weighted_mean(bv) # 指数重み平均（末尾重視）
    return b_full, b_early, b_mid, b_late, b_wls
```

6地層（ANCC, ASTNU, ASTNL, EGFDU, EGFDL, BUDA）× 5バリアント = 30 b_well 特徴量。
